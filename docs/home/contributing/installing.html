Install a local copy of the Theme to develop custom components:

Development Requirements
  • NodeJS - version 0.12.4 or above
  • Bower - npm install -g bower
  • Grunt-CLI - npm install -g grunt-cli
Installation Instructions
  1. [Fork then] clone the repository
    • git clone git@github.com:<your-user-name>/theme.git
  2. Install the NPM dependencies
    • npm install
  3. Install the Bower dependencies
    • bower install
  4. Run the build task and start the server
    • grunt
  5. Open the local built Theme docs in a web browser
    • http://127.0.0.1:3001
